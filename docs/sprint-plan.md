# 14-Day Sprint Plan (MVP)

1. Setup monorepo, linting, env management.
2. Build frontend shell (hero, product, story, shop, checkout placeholder).
3. Add React Three Fiber scene and placeholder models.
4. Add GSAP scroll-trigger sequence.

7. Implement cart + order creation endpoints.

9. Integrate frontend cart/auth/checkout to backend APIs.
10. Build user dashboard (orders, profile).
11. Add admin-lite product/order management endpoints.
12. Optimize 3D assets and lazy loading.
13. Add telemetry + error handling.
14. Deploy web + api and run production smoke tests.
